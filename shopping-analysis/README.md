# Shopping Dataset Analysis

## Objective
To perform data cleaning, exploration, and basic analysis using Python Pandas on a shopping dataset.

## Dataset Used
combined_dataset.csv (Kaggle Shopping Dataset)

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Steps Performed

1. Loaded dataset using Pandas
2. Checked data shape, columns, and types
3. Handled missing values
4. Removed duplicate records
5. Converted price columns to numeric
6. Created new features:
   - price_difference
   - popularity_metric
   - total_amount
7. Performed analysis and visualization

## Visualizations
- Histogram (price distribution)
- Bar chart (category analysis)
- Boxplot (price comparison)
- Scatter plot (rating vs price)

## Insights
- Ratings affect product popularity
- Some categories are more expensive
- Discounts impact customer behavior
- High-rated products get more attention

## Output Files
- cleaned_dataset.csv
- analysis.ipynb
